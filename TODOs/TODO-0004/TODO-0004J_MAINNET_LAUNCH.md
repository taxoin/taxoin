# TODO-0004J: Mainnet Launch Requirements

## Сколько нод нужно для mainnet?

### Теория: BFT требует 3f+1

```
f = количество византийских (злых/сломанных) валидаторов
n = общее количество валидаторов
quorum = 2f+1

Условие: n >= 3f+1

f=0 → n >= 1  (нет защиты)
f=1 → n >= 4  (устойчивость к 1 злому)
f=2 → n >= 7  (устойчивость к 2 злым) ← НАШ ТЕКУЩИЙ ПАРАМЕТР
f=3 → n >= 10 (устойчивость к 3 злым)
```

### Варианты для mainnet

| Нод | Выдерживает | Риск | Genesis | Статус |
|-----|------------|------|---------|--------|
| **1** | Ничего | Любая проблема = сеть умерла | ✅ | 🚫 Не mainnet |
| **3** | 0 злых (f=0) | 1 злой валидатор ломает сеть | ✅ 3/7 аттестации | 🟡 Testnet OK |
| **4** | 1 злой (f=1, quorum=3) | 2 злых = катастрофа | ⚠️ Нужно 3/4 | 🟢 Min viable |
| **5** | 1 злой (f=1, quorum=3) | Запас прочности +1 | ⚠️ Нужно 3/5 | 🟢 Хорошо |
| **7** | 2 злых (f=2, quorum=5) | Классический BFT | ✅ 3/7 | 🟢 Цель |
| **12** | 3 злых (f=3, quorum=7) | Высокая надёжность | ❌ Нужно менять | 🔵 Будущее |
| **22** | 7 злых (f=7, quorum=15) | Как Cosmos | ❌ Нужно менять | 🔵 Далеко |

### А если я один хочу запустить mainnet?

```
Можно: 1 человек = 7 контейнеров на 7 разных VPS
Плохо: single point of failure (вы = цель)
Лучше: найти 3-6 друзей/команд, who trust each other

Главный вопрос mainnet не технический, а СОЦИАЛЬНЫЙ:
  "Кому доверяют валидацию?"
```
