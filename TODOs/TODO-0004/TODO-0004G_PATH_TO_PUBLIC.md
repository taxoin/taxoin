## Что минимально нужно для "скачал-поставил-работает"

### Уровень 1: Инфраструктура (DevOps)

**Сейчас:** `python3 src/cli.py` — нужно клонировать репозиторий, установить
зависимости, разобраться с git.

**Нужно для публики:**

- [ ] **Docker Compose** — одна команда `docker compose up` поднимает сеть:
  ```
  3 валидатора (+ load balancer)
  1 API gateway (JSON REST)
  1 bootstrap seed
  ```
- [ ] **pip install taxoin** — пакет в PyPI
- [ ] **One-click deploy** на Railway / Fly.io / Render — для тех, кто не шарит
- [ ] **Public testnet** — dev-сеть, где можно получить testnet Ⓣ

---

### Уровень 2: API для внешних сервисов (Developers)

**Сейчас:** Нет API. Только Python-вызовы из CLI.

**Нужно:**

- [ ] **REST API** (FastAPI or similar):
  ```bash
  POST /api/wallet          # создать кошелёк
  GET  /api/balance/{addr}  # проверить баланс
  POST /api/tx/send         # отправить Ⓣ
  GET  /api/services        # список услуг
  POST /api/service/register  # зарегистрировать услугу
  ```
- [ ] **Webhook callback** — бизнес получает уведомление о платеже:
  ```json
  POST /webhook { "tx_id": "...", "amount": 1.0, "from": "0x...", "status": "confirmed" }
  ```
- [ ] **API key authentication** — для интеграций с внешними сервисами
- [ ] **OpenAPI/Swagger docs** — чтобы любой разработчик понял за 5 минут

---

### Уровень 3: Кошелёк для людей (UI/UX)

**Сейчас:** Терминал. `taxoin wallet address`, `taxoin tx send`.

**Нужно:**

- [ ] **Web-кошелёк** (SPA на React/Vue):
  - Показать баланс
  - Отправить Ⓣ по QR или адресу
  - История транзакций
  - Создание кошелька (скачать приватный ключ)
  - Сканировать QR для оплаты

- [ ] **Telegram Mini App** — кошелёк прямо в Telegram:
  - Самый быстрый путь к массам (нет установки)
  - `/start` → создаёт кошелёк
  - Отправить Ⓣ контакту
  - Платить сервисам

- [ ] **Бизнес-дашборд**:
  - Принимать Ⓣ (QR для клиентов)
  - История платежей
  - Отчёт за день/неделю/месяц
  - Выгрузка для бухгалтерии

---

### Уровень 4: Интеграция с реальными бизнесами

**Для таксопарка:**
```
Клиент: оплата Ⓣ через QR в машине
Водитель: получает Ⓣ на кошелёк
Таксопарк: комиссия 5% в Ⓣ
→ Всё через Telegram Mini App
```

**Для парикмахерской:**
```
Мастер: регистрирует услугу "стрижка" за 0.5 Ⓣ
Клиент: оплачивает через QR на визитке
→ Автоматическая запись в Service Registry
→ +1 репутация мастеру
```

**Для биржи труда:**
```
Работодатель: создаёт задание за 10 Ⓣ
Работник: выполняет → mutual attestation
→ Escrow через Balance Hold
→ Споры через Dispute Resolution
```
