# TODO-0005B: Web Dashboard для Валидаторов

**Цель:** Веб-интерфейс для мониторинга и управления нодой.
Зелёная/красная лампочка, графики, кнопки.

## Страницы

```
/dashboard
├── Статус сети    — блоки, валидаторы, TPS, зелёная лампочка
├── Валидаторы     — список, кто онлайн, кто офлайн
├── Disputes       — открытые споры с кнопками голосования
├── Genesis        — очередь новых людей на аттестацию
├── Балансы        — топ кошельков
└── Логи           — последние события сети
```

## Технически

```html
<!-- Простая SPA на React/Vue, или вообще на чистом HTML + JS
     Подключается к API localhost:47780 -->

<div class="status-card">
  <div class="indicator {{network.status === 'ok' ? 'green' : 'red'}}"></div>
  <span class="blocks">{{network.blocks}} blocks</span>
  <span class="validators">{{network.validators}} validators</span>
</div>
```

Можно вообще без фреймворка — на чистом HTML/CSS/JS.
У нас уже есть `web/wallet.html` — по тому же принципу.

## Что должно быть на dashboard

```bash
# Все данные уже есть в API:
GET /api/status           → блоки, валидаторы
GET /api/validators       → список
GET /api/disputes         → споры (TODO-0004B)
GET /api/reputation/{addr} → репутация
GET /api/services         → услуги
```
