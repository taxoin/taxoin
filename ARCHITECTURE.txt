                           GIT AS BLOCKCHAIN MAPPING
═══════════════════════════════════════════════════════════════════════════════

┌──────────────────────┬────────────────────────────────────────────────────┐
│   Blockchain         │   Git                                              │
├──────────────────────┼────────────────────────────────────────────────────┤
│ Block                │ Commit                                             │
│ Block hash           │ Commit SHA-1                                       │
│ Parent hash          │ Parent commit SHA                                  │
│ Merkle root          │ Tree SHA (contains block data)                     │
│ Timestamp            │ Commit timestamp                                   │
│ Block data           │ Tree object (JSON file)                            │
│ Chain                │ Branch / Tag history                               │
│ Genesis block        │ Root commit (no parent)                            │
│ Chain head           │ refs/tags/chain-head                               │
│ Mining               │ Finding commit with N leading zeros                │
│ Verification         │ git log --verify (check parent links)              │
└──────────────────────┴────────────────────────────────────────────────────┘

Example Git Structure:

.taxoin/
└── .git/
    ├── objects/
    │   ├── 00/
    │   │   └── 00a3f2b1c... (Block #1 - genesis)
    │   ├── 00/
    │   │   └── 00b4e3c2d... (Block #2)
    │   └── 00/
    │       └── 00c5f4d3e... (Block #3)
    ├── refs/
    │   └── tags/
    │       └── chain-head -> 0000c5f4d3e...
    └── HEAD -> refs/tags/chain-head

│   - Transaction inputs/outputs  │   - Nonce for replay protection         │

│ • Proof of Work                 │ • Gas System                            │
│   - SHA256 mining               │   - gas_price × gas_limit               │
│   - Difficulty adjustment       │   - Fees go to miner                    │
│                                 │                                         │
│ • ECDSA secp256k1               │ • Async Transactions                    │
│   - Same curve as Bitcoin       │   - Mempool queue                       │
│   - Address derivation          │   - Concurrent submission               │
│                                 │                                         │
│ • Coinbase Reward               │ • State Snapshot                        │
│   - 50 coins per block          │   - Account balances in block           │
│   - Goes to miner               │   - State transitions                   │
└─────────────────────────────────┴─────────────────────────────────────────┘
