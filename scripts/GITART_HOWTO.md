# GITART — Pixel Art в GitHub/GitLab Contribution Graph

## Что это такое

GitHub и GitLab показывают на профиле «тепловую карту» активности —
сетку из зелёных квадратиков (7 строк × 52 колонки = 364 дня).
Каждый квадратик = один день. Чем больше коммитов в тот день — тем темнее.

**Идея:** создать бэкдейтированные коммиты на конкретные даты так,
чтобы квадратики сложились в буквы. Результат — надпись «TAXOIN»
видна на странице профиля как пиксельная графика.

## Предварительные требования

1. **Пустой репозиторий** на GitHub/GitLab (создать руками через веб-интерфейс)
2. **Personal Access Token (PAT)** с правами `repo`
3. **Email** в коммитах должен совпадать с verified email на аккаунте

### Получить GitHub PAT

```
GitHub → Settings → Developer settings
→ Personal access tokens → Tokens (classic)
→ Generate new token → галка [✓] repo → Copy
```

Токен выглядит так: `ghp_xxxxxxxxxxxxxxxxxxxx`

## Запуск

### Шаг 1 — Предпросмотр (без коммитов)

```bash
python scripts/gitart.py --preview
```

### Шаг 2 — Создать коммиты в локальную папку

```bash
mkdir -p /tmp/taxoin-art

python scripts/gitart.py \
  --remote https://github.com/ТВОЙ_ЮЗЕР/РЕПО.git \
  --email  твой@email.com \
  --name   "Твоё имя" \
  --dir    /tmp/taxoin-art \
  --dow    0              # 0=воскресенье (GitHub), 1=понедельник (GitLab EU)
```

### Шаг 3 — Запушить с токеном

```bash
git -C /tmp/taxoin-art push \
  https://ТВОЙ_ЮЗЕР:ВАШ_TOKEN@github.com/ТВОЙ_ЮЗЕР/РЕПО.git \
  main --force
```

### Шаг 4 — Проверить

Открой `https://github.com/ТВОЙ_ЮЗЕР` — через 5–10 минут граф обновится.

## Параметры скрипта

| Параметр | По умолчанию | Описание |
|----------|-------------|----------|
| `--remote` | — | URL репозитория (обязательный) |
| `--email` | — | Email аккаунта (обязательный) |
| `--name` | `TAXOIN` | Имя автора в коммитах |
| `--branch` | `main` | Ветка (должна быть дефолтной в репо) |
| `--intensity` | `4` | Коммитов на пиксель: 1=светлый, 4=тёмный |
| `--dow` | `0` | Первый день недели: 0=Sun, 1=Mon |
| `--dir` | авто temp | Рабочая папка (авто удаляется после push) |
| `--preview` | — | Только ASCII-превью, без коммитов |

---

## Важные детали

### Email должен совпадать
GitHub проверяет email автора коммита.
Если email не совпадает с аккаунтом — квадратики не засчитываются.

### --dow подбирается под платформу
- **GitHub** — начало недели в воскресенье (`--dow 0`)
- **GitLab** (EU-локаль) — начало недели в понедельник (`--dow 1`)

Если картинка смещена на 1 строку — попробуй другое значение `--dow`.

### Граф показывает последние 52 недели
Скрипт автоматически считает диапазон дат от текущей даты назад на год.
Коммиты вне этого окна невидимы на графе.

### Дублирующийся запуск
Если репо уже существует на сервере с другим содержимым — `--force` затирает его.
Скрипт всегда делает force push, так что репо `taxoin-art` лучше держать
только для арта (не смешивать с рабочим кодом).

## Пример реального запуска (taxoin на GitHub)

```bash
# Создали: https://github.com/taxoin/taxoin

mkdir -p /tmp/taxoin-art

python scripts/gitart.py \
  --remote https://github.com/taxoin/taxoin.git \
  --email  taxoinx@gmail.com \
  --name   "taxoin" \
  --dir    /tmp/taxoin-art \
  --dow    0

# После завершения (360 коммитов) — пушим с PAT:
git -C /tmp/taxoin-art push \
  https://taxoin:ghp_ВАШ_ТОКЕН@github.com/taxoin/taxoin.git \
  main --force
```

Результат: https://github.com/taxoin — надпись TAXOIN в contribution graph.

## Как изменить надпись

В `scripts/gitart.py` поменяй константы вверху файла:

```python
WORD = "TAXOIN"   # ← сюда новое слово

FONT = {          # ← сюда добавь новые буквы (7 строк × 5 колонок)
    "T": [...],
    ...
}
```

Каждая буква — матрица 7×5, где `1` = тёмный квадратик, `0` = пустой.
